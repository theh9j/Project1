# Bottle Fill Shader Graph

A configurable bottle fill Shader Graph with color bands and height masking.

## Overview
Includes ShaderGraph assets, sample scenes, and documentation for creating layered bottle fill visuals.

## Features
- Height-based fill masking with adjustable fill amount.
- Four band colors with adjustable band count.
- Band scale control for layer spacing.
- Alpha clipping ready for sprite masks.
- Sample scenes for multiple bottle silhouettes.

## Requirements
- Unity 2021.3 or later
- Shader Graph package installed

## Quick Start
1. Import the package into your project.
2. Open the sample scene to preview variations.
3. Assign the provided material to a SpriteRenderer.
4. Adjust material properties (colors, fill amount).
